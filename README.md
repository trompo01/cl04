# EL PUBLICO OLVIDADO DEL MUNDIAL 2026

**SOBRE QUE SABER DE FUTBOL CUANDO EL MUNDIAL, LA FIFA y LA CONMEBOL TE IMPORTAN CERO - * ACTUALIZACION DIARIA ***

Si no sabes los colores de las camisetas, ni los nombres de los jugadores o si preferís ver una serie o leer un libro mientras se juega una final, este es tu lugar. El deporte de los ilegales con Inmigración en el banco de suplentes esperando su momento como un cocodrilo en una piscina pública.

## Características
- Diseño minimalista, fondo oscuro, letras blancas, alto contraste
- 7 páginas en un solo archivo `index.htm` (SPA)
- Página 1: tabla 2x3 con 6 botones 3D cuadrados con imágenes
- Páginas 2-4: fondo con bandera, visor de video pCloud y visor DOCX nativo desplazable
- Página 5: libros
- Página 6: video del día
- Página 7: Estados Unidos + links
- Efecto 3D tilt real al mover el mouse, animación float suave
- Totalmente responsive para Android y iPhone
- Botones con texto alternante (VIDEO/DEL/DIA/CLICK AQUI, etc.)

## Archivos incluidos

## Cómo publicar en GitHub Pages
1. Crea un repo nuevo: `mundial2026-olvidado`
2. Sube todos los archivos a la rama `main`
3. Settings → Pages → Deploy from branch → main / root
4. Tu sitio estará en `https://tuusuario.github.io/mundial2026-olvidado/`
